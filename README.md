# A股每日投资分析报告

自动生成 A 股每日投资分析报告，包含大盘总览、板块热点、个股异动、精选标的推荐、财经要闻及市场综述。

## 功能特性

- 📊 **大盘总览** — 7大指数行情、市场情绪、两市成交额
- 🔥 **板块热点** — 行业板块涨幅排行、概念板块分析
- 🎯 **个股异动** — 涨跌幅前列个股
- ⭐ **精选标的** — 基于量化模型的 TOP5 个股推荐 + ETF 精选
- 📰 **财经要闻** — 当日重要资讯汇总
- 📝 **市场综述** — 策略建议与关键信号

## 运行方式

### 自动运行（GitHub Actions）

每日北京时间 20:00（周一至周五）自动生成报告。

### 手动运行

```bash
pip install -r requirements.txt
python report.py
```

报告将生成在 `/workspace/A股投资分析报告_YYYY-MM-DD.md`。

## 精选标的筛选逻辑

| 维度 | 权重 | 说明 |
|------|------|------|
| PE估值 | 25分 | 5-15倍最优 |
| PB估值 | 20分 | 0.5-2.5倍安全边际高 |
| 安全边际 | 20分 | 距52周高点回撤 |
| 市值规模 | 15分 | 200-3000亿适中 |
| 技术面 | 20分 | 趋势强度+涨幅合理性 |

### 硬性过滤条件
- 股价 < 100元
- 非科创板（688开头）
- 非 ST 股
- PE/PB 有效（盈利企业）

## 数据源

- **腾讯财经** — 实时行情、PE/PB/市值
- **东方财富** — 板块排行、资金流向
